# 検索/タグ絞り込み/口コミ投稿の機能化

## Goal
- 親ページのヒーロー文言更新と検索機能の実装
- 企業ページのタブ(タグ)絞り込み・検索・口コミ投稿を動作させる
- 口コミ投稿をSupabaseに保存できるようにする

## Done
- 親ページのヒーロー文言を差し替え、GETクエリで企業検索を実装
- 企業ページのタブ切替時にタグ/検索条件をAPI経由で再取得
- 検索フォームを実装し、記事検索をAPIに接続
- 口コミ投稿フォームを追加し、POST /api/articles で保存
- 記事APIにタグ・検索・投稿対応を追加

## Discoveries
- RLSが読み取り専用のため、INSERT許可がないと投稿が失敗する

## Decisions
- タブ切替はクライアント側の絞り込みではなくAPI再取得で対応
- 検索はGETクエリ/POST APIに寄せて最小実装

## Notes
- Supabase MCPは未接続のため既存クライアントで実装
